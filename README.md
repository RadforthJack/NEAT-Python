# NEAT-Python

Forked from [SirBob01/NEAT-Python](https://github.com/SirBob01/NEAT-Python) and extended for my MEng Computer Science dissertation at the University of York.

## About

This library implements NeuroEvolution of Augmenting Topologies (NEAT) — an algorithm that evolves both the weights and structure of neural networks over generations.

## Dissertation Project

The library was modified and used to simulate a predator-prey environment, where populations of predator and prey agents evolved independently using NEAT. The project analysed emergent behaviours arising from co-evolution, with results evaluated using statistical analysis.

The main modification made to the project was to give it multi-threading capabilities, to increase the number of simulations that could be run in the available time
